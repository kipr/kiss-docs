Java Programming/Files and Streams

Files and Streams are an important part of Java, there are used to store information on any computer it is run on. they are used by most programs to remeber data for uses such as creating document, saving user preferences, or running macros.

Text Files

These are used to save information as text. this has several atvantages. it can be understood by any program designed to use the stored information, can read or write xml data (although there are other methods speciffical designed for xml). some disadvantages are that they are slower adn require much more space.
